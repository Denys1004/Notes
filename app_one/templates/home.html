HOW DOES IT WORK?

Create Topic

Here you can create a topic for which you will write notes. The number of topics is unlimited. Each topic has its own page for creating notes where you can see notes that are already created for that topic. All topics will be displayed on the sidebar for easier navigation and search.

Add Notes

You can write notes on each topic. Each note can contain a title that will be displayed in the sidebar with a link to this note. To find the note in a specific topic click on the down arrow next to the topic name (you'll see it when topic has at least one note), it will open up a list of the notes below.

Manage

You can add new topics and delete old ones. You can edit the notes, modify them, and delete them as needed. A note can contain pictures and videos (you can find these options in a text editor). The number of topics and notes is unlimited. Enjoy!
